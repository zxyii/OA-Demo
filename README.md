 # OA 系统

### 项目介绍 + 技术栈
* ​整体框架结构基于**SpringBoot**  +  **MyBatis**
* ​数据库：**MySql**、**少许**Redis
* ​登陆及鉴权部分：使用**sa-token**框架                                                 

##### 关于项目
* ****：统一返回结果Result
* **exception**： 全局异常处理GlobalExceptionHandler
* **config**：sa-token路由拦截器、返回权限码、Redis配置类(自定义RedisTemplate的序列化方式)
* **properties**：Minio参数
